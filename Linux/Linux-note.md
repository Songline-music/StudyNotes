# Linux 笔记

> 常用 Linux 命令与快捷键整理。

## 常用指令

### 1. `ls`

显示当前文件
- `-l`：显示详细内容
- `-a --all`：显示隐藏内容
- `-h`：显示文件大小
- `-t`：按照时间顺序排列
- `-r --reverse`：逆序排列
- `-i`：显示文件的inode

### 2. `clear`

清除屏幕显示

### 3. `rm`

删除文件
- `-i`：向用户询问是否删除
- `-f`：文件强制删除
- `-r`：递归删除文件(rm -rf)((awa))

### 4. `history`

显示所有历史指令, ! + 行数 执行指定的历史指令

### 5. `pwd`

显示当前目录的路径

### 6. `which 指令名`

查看命令的可执行文件所在路径

### 7. `cd`

进入文件
- `cd /`：跳转到根目录
- `cd ~`：跳转到home目录
- `cd ..`：跳转到上级目录
- `cd ./home`：跳转到当前目录的home目录下
- `cd /home/lion`：跳转到根目录下的home目录下的lion目录
- <cd /ho单次 tab键会自动补全路径 + 两次 tab键会列出所有可能的目录列表>

### 8. `du`

列举目录大小信息
- `-h`：显示文件大小
- `-a`：展示所有信息
- `-s`：只显示总计大小

### 9. `cat`

一次性显示文件的所有内容
- `-n`：显示行号

### 10. `less`

分页显示文件内容
- 快捷键:
- `空格`：前进一页
- `b`：后退一页
- `y`：后退一行
- `↑↓`：回退或前进一行
- `d`：前进半页
- `u`：后退半页
- `q`：停止读取文件
- `=`：显示当前页面的内容在文件中的行以及其他信息
- `h`：显示帮助页面
- `/`：进入搜说模式 --> n 跳到一个符合的项目 N 跳到上一个符合的项目

### 11. `head`

显示文件的开头10行
- `-n + 数字`：指定行数

### 12. `tail`

显示文件的结尾10行
- `-n + 数字`：指定行数
- -f -s + 数字 每过(数字)秒检查文件是否有更新

### 13. `touch`

创建文件

### 14. `mkdir`

创建一个目录
- `-p`：递归的创建目录结构

### 15. `cp 目标文件 拷贝出来的文件`

拷贝文件和目录
- `cp 目标文件 目标目录`：拷贝文件到指定目录下,文件名相同
- `cp 目标文件 目标目录/拷贝出的文件名`：拷贝文件到指定目录下,文件名为指定名
- `cp *.txt 目录名`：把当前目录下的所有txt文件拷贝到指定目录下
- `-r`：递归拷贝,拷贝一整个目录

### 16. `mv`

移动文件或目录(方式同cp)
- `mv 原文件名 新文件名`：文件重命名

### 17. `ln`

创建链接
- `ln file1 file2`：创建file1和file2的硬链接
- `ln -s file1 file2`：创建file1和file2的软链接

### 18. `sudo 指令`

以root身份运行命令

### 19. `useradd`

添加新用户

### 20. `passwd`

修改用户密码

### 21. `userdel`

删除用户

### 22. `su 用户名`

切换用户

### 23. `groupadd 群组名`

创建群组

### 24. `groupdel 群组名`

删除群组

### 25. `groups 用户名`

查看用户所在群组

### 26. `usermod`

修改用户的账户信息
- -l 对用户重命名
- -g 修改用户所在群组
- -G 退出当前群组,一次性让用户添加多个群组
- -a -G 追加群组

## 快捷方式

- `^R`：用于查找使用过的命令
- `^L`：清除屏幕
- `^C`：终止正在执行的指令
- `^U`：从光标位置剪切到行首
- `^K`：从光标位置剪切到行尾
- `^W`：剪切光标左侧的一个单词
- `^Y`：粘贴
- `^A`：光标跳到命令行开头
- `^E`：光标跳到命令行结尾
- `^D`：关闭Shell会话
